FlashTools AE_FAIO IO port access assistant

本ソフトウェアは、東芝およびフィックスターズの公式ソフトウェアではありません

お問い合せは、@Seg_faulまでお願いします。


・前提
このツール群は、秋月ＦｌａｓｈＡｉｒ　ＤＩＰ　ＩＯボードキット(K-10007 AE-FAIO)のIOポート操作をWi-Fiから簡単にできるようにしたものです。
ＦｌａｓｈＡｉｒ　ＤＩＰ　ＩＯボードキット(K-10007 AE-FAIO)に差し込まれた状態でのみ正しく動作します。
このhelp.htmは、Wi-Fi経由でお読みください。

・配置

上の画像(favion.ico)が表示されるように、ルートに配置してください。
http:/flashair/AE_FAIO_HTTP_IO/help.htm　でこのファイルが閲覧できるように配置してください。

・使い方
まず、自動セットアップを実行します。
http:/flashair/AE_FAIO_HTTP_IO/setup.lua

手動でセットアップする場合は、/SD_WLAN/CONFIGを開き、IFMODE=1を追記してください。(設定後、FlashAirの再接続が必要です。)

その後、以下のリンクを踏むことで、動作を確認できます。
お好みのWebページからリンクやボタンを設置しても利用できますし、javascriptからXmlHttpRequestを用いてアクセスしても動作します。
※STAモードでご利用時は、flashairの部分をIPアドレスに置き換えてご利用ください。

IOポート出力
http:/flashair/AE_FAIO_HTTP_IO/digitalWrite/0/HIGH.lua
http:/flashair/AE_FAIO_HTTP_IO/digitalWrite/0/LOW.lua
http:/flashair/AE_FAIO_HTTP_IO/digitalWrite/1/HIGH.lua
http:/flashair/AE_FAIO_HTTP_IO/digitalWrite/1/LOW.lua
http:/flashair/AE_FAIO_HTTP_IO/digitalWrite/2/HIGH.lua
http:/flashair/AE_FAIO_HTTP_IO/digitalWrite/2/LOW.lua
http:/flashair/AE_FAIO_HTTP_IO/digitalWrite/3/HIGH.lua
http:/flashair/AE_FAIO_HTTP_IO/digitalWrite/3/LOW.lua

IOポート入力
http:/flashair/AE_FAIO_HTTP_IO/digitalRead/0.lua
http:/flashair/AE_FAIO_HTTP_IO/digitalRead/1.lua
http:/flashair/AE_FAIO_HTTP_IO/digitalRead/2.lua
http:/flashair/AE_FAIO_HTTP_IO/digitalRead/3.lua

LCD出力
http:/flashair/AE_FAIO_HTTP_IO/lcd/clear.lua
http:/flashair/AE_FAIO_HTTP_IO/lcd/print.lua?Hello%0AFlashAir
http:/flashair/AE_FAIO_HTTP_IO/lcd/print.lua?Hey

・応答
基本的に、要求に応じた応答が帰ってきます。
帰ってこない場合は、ファイルが破損しているか、ディレクトリ配置のミスです。
(ファイル参照の関係上、ルートディレクトリに正しく配置されていないと動作に失敗します。)
